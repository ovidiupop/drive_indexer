Drives

In the Drives tab you have to set active the drives which are connected to your computer.

From the list you select the drive then click Add.
The new added drive will be included in the table, and you can set for it your Own Label to easily identify it.

To edit a field, you have to use the form and to save new data, you have to press Save Changes else the changes will not be saved to database.

If you plug a new drive to the computer, application will be notified and will automatically add the new drive to the list of drives. Will be the last one in the list. You can add it if you wish to index it!

If you unplug one drive added to the table, that will be marked in the Active column with a red X and will be removed from the list of drives. The indexed folders belonging to this drive will not be visible anymore in the Folder tab until the drive will be re-connected. The indexed files will obviously be available for search.

If you remove a drive from the table (do this only if you will not reuse that drive), the indexed files will remain available for search. If you wish to remove them, you have to delete indexed folders which belong to that drive.
